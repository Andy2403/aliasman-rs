Maneja tus alias de forma fácil y rapida.

## Cómo usarse

El funcionamiento de AliasMan se basa en 4 comandos:

1. Add
```shell
aliasman add hello echo "Hello World"
```
2. Remove or Rm
```shell
aliasman remove hello
aliasman rm hello
```
3. Replace
```shell
aliasman replace hello echo "Hello!"
```
4. List
```shell
aliasman list
```
